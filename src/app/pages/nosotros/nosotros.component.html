Objetivos

El objetivo del proyecto es desarrollar una aplicación web funcional que permita a los usuarios interactuar con el contenido del foro de manera eficiente y efectiva.
un lugar donde puedan compartir documentos de estudio, resumenes y otros materiales educativos, tambien un lugar donde puedan aclarar dudas o hacer preguntas entre compañeros.
otro de los objetivos que teniamos con este proyecto era ya que teniamos otro trabajo de otra materia "base de datos" una idea fue unificar los dos trabajos en uno para que el foro tuviera una base de datos propia usando las herramientas que aprendiamos en esa materia lamentablemente ese objetivo no se llevo a cabo por cuestiones de tiempo y conocimientos.

Contexto

El proyecto nace apartir de una de las tareas de una materia del primer cuatrimestre del segundo año en la materia de programacion web, tuvimos varias propuestas de proyectos y el que escojio la profe fue este de un foro, propusimos este proyecto porque nos parecio interesante y nos gustaria poder desarrollarlo y aprender de ello.
Este proyecto se desarrollo en un tiempo de 3 meses aproximadamente, y se fue desarrollando a medida que avanzaba la materia, con la colaboracion los miembros del grupo.
El que se puede ver aqui en la pagina seria como un prototipo de lo que seria el la idea final, ya que no se pudo desarrollar todo el proyecto por falta de tiempo y conocimientos, pero se logro desarrollar una parte importante del proyecto.
